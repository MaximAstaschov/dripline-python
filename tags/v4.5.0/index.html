Welcome to Dripline-python’s documentation!¶

Dripline-python builds on dripline-cpp to provide useful implementations for assembly of a dripline mesh. The package is split into three components (each a namespace): core, for extending dripline-cpp with a pythonic interface and adding the most basic and generic features; implementations, for ready to use classes that enable generic support for various devices, resources, and behaviors; and extensions, for enabling plugins via namespace packages. There is also a commandline tool, dl-serve, which enables YAML-based configurations to define the components in a running service.
